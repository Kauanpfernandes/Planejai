# 💰 Planej.ai - Educador Financeiro Inteligente

## 📖 Sobre o Projeto

O Planej.ai é uma proposta de aplicação web desenvolvida durante o desafio da DIO, com o objetivo de auxiliar usuários no planejamento financeiro pessoal através de Inteligência Artificial Generativa.

A plataforma foi idealizada para oferecer orientações financeiras personalizadas, permitindo que o usuário informe sua realidade financeira e receba análises, recomendações e sugestões para alcançar seus objetivos de forma organizada e consciente.

---

## 🎯 Objetivo

Desenvolver um Educador Financeiro Inteligente capaz de:

* Auxiliar no controle financeiro pessoal;
* Incentivar a criação de metas financeiras;
* Apresentar diagnósticos personalizados;
* Sugerir formas de economia;
* Recomendar alternativas para geração de renda extra;
* Utilizar Inteligência Artificial para gerar insights financeiros.

---

## 👥 Público-Alvo

O sistema foi pensado principalmente para:

* Jovens iniciando a vida financeira;
* Universitários;
* Pessoas que desejam organizar melhor suas finanças;
* Usuários que possuem metas financeiras específicas;
* Pessoas interessadas em educação financeira prática.

## 🚀 Funcionalidades Propostas

### 📋 Formulário Inteligente

Coleta informações como:

* Renda mensal;
* Gastos fixos;
* Dívidas e parcelas;
* Meta financeira;
* Valor desejado;
* Prazo para alcançar a meta.

---

### 📊 Simulação Financeira

Realiza cálculos automáticos para:

* Economia mensal disponível;
* Valor necessário para atingir a meta;
* Viabilidade do objetivo financeiro;
* Planejamento financeiro personalizado.

---

### 🤖 Inteligência Artificial

Utiliza IA Generativa para fornecer:

* Diagnóstico financeiro;
* Sugestões de economia;
* Recomendações de investimentos;
* Ideias para renda extra;
* Mensagens motivacionais personalizadas.

## 🔄 Fluxo de Funcionamento

1. O usuário inicia uma nova simulação.
2. Informa seus dados financeiros.
3. Define sua meta financeira.
4. O sistema realiza cálculos automáticos.
5. A IA analisa os dados.
6. O usuário recebe um diagnóstico financeiro personalizado.
